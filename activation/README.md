4. **Layer Norm**
   - 公式：`y = (x - μ) / √(σ² + ε) * γ + β`
   - 需要对每个样本的特征维度进行归一化
> rsqrtf 是开方后倒数
计算步骤假设输入是一个向量 $x = [x_1, x_2, ..., x_H]$，其中 $H$ 是特征维度（Hidden size）。

第一步：计算均值 $\mu$计算这一个样本所有特征的平均值：$$\mu = \frac{1}{H} \sum_{i=1}^{H} x_i$$第二步：计算方差 $\sigma^2$计算这一个样本特征的离散程度：$$\sigma^2 = \frac{1}{H} \sum_{i=1}^{H} (x_i - \mu)^2$$第三步：归一化 (Normalization)将原始特征缩放到均值为 0，方差为 1 的标准分布。为了防止分母为 0，会加上一个极小值 $\epsilon$：$$\hat{x}_i = \frac{x_i - \mu}{\sqrt{\sigma^2 + \epsilon}}$$第四步：缩放和平移 (Scale and Shift)为了让模型保留表达能力（万一 0-1 分布不合适呢？），引入两个可学习的参数 $\gamma$ (gain) 和 $\beta$ (bias)：$$y_i = \gamma \hat{x}_i + \beta$$
